Migration plan

  1. Add .dark tokens to src/styles/global.css (navy surface, muted text, blob colors).
  2. Create src/components/BackgroundBlobs.astro with SVG + randomizing script.
  3. Create src/components/ThemeToggle.astro (persists in localStorage, sets class="dark" on <html>).
  4. Replace motion-aurora-shell classes with apu-section-shell + <BackgroundBlobs seed="section-name" /> in every listed section.
  5. Remove old aurora/grid keyframes and variant classes from global.css.

Open decisions

  • Where should the theme toggle live? Nav bar, footer, or a floating button?
  • How many blobs per section? Default 5 OK?
  • Should reduced-motion/transparency users get a flat solid fallback?
